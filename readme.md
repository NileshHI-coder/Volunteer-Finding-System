# 🤖 AI Volunteer Matching System

## 📌 Overview
The AI Volunteer Matching System is a web-based platform that connects people in need with suitable volunteers using Artificial Intelligence.

Users can submit help requests, and volunteers can register with their skills and availability. The system uses AI (Google Gemini) to match the best volunteers to each request based on skills, location, and urgency.

---

## 🚀 Features
- 📝 Submit Help Requests
- 🙋 Volunteer Registration
- 🤖 AI-Based Volunteer Matching
- 📊 Dashboard for Requests, Volunteers & Matches
- 🔥 Real-time data using Firebase Firestore

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Python (Flask)
- **Database:** Firebase Firestore
- **AI Integration:** Google Gemini API

---

## ⚙️ How It Works
1. User submits a help request
2. Volunteers register with their skills
3. AI analyzes request & volunteers
4. Best matches are generated
5. Dashboard displays results
